Просмотр версии PAX D200

Просмотр версии прошивки:

Открыть папку

C:\PC_STUB

Открыть на правку файл

"pinpad.ini"

раскомментировать (убрать знак ; ) строчку

;ShowScreens=1

Добавить строчку (если нет такой)

PrinterFile=p

Сохранить и закрыть файл

Запустить файл

"Loadparm.exe"

В появившемся всплывающем окне "UPOS.Главное меню", выбрать

"Операции" -"Помощь"

Программа закроется и создаст файл "p"

Открыть файл "р" через Notepad++. В файлике будет указана прошивка модуля, версия операционной системы

Обязательно заново отредактировать файл "pinpad.ini" закоментировать строку

;ShowScreens=1

Сохранить и закрыть файл
