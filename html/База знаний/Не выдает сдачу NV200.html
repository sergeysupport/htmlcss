Не выдает сдачу NV200

Не выдает сдачу после оплаты:

  1. Проверить подключение и опросить купюрник из служебного меню
  2. Проверить наличие сдачи на ленте
  3. Проверить индикаторный цвет принимающей части (красный/синий) - если моргает красно-синим, устранить причину
  4. Сделать тестовую выдачу через системное меню
  5. Снять лог на предмет ошибки купюроприемника
  6. При ошибке SmartPayOut-Detected error with payout device:
При помощи програмы ITL включить/выключить/перезагрузить устройство
Рекомендовать скинуть всю сдачу в кассету (делает клиент сам) и загрузить заново
Рекомендовать по инструкции выполнить чистку устройства
  1. При повторении случаев невыдачи, предложить отправить устройство на сервисную чистку (платно)
  2. В случае замятия (по логу либо по цвету принимающей части), по инструкции достать купюру.
  3. В случае, если при оплате сдача не выдается, а из служебного меню выдает, проверить подключение - должно быть подключено в USB 2.0
  4. При отсутствии подключения в слжебном меню, выкл/вкл порт в панеле управления (диспетчер устройств), через ITL многократно сделать Reset, пока не появится устройство
