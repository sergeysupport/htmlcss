Не выдает сдачу Hopper

  1. Проверить подключение и опросить Хоппер из служебного меню
  2. Проверить наличие сдачи
  3. Сделать тестовую выдачу монет через системное меню
  4. Снять лог на предмет ошибки монетоприемника
  5. Если по логу устройство отбивается по таймауту через какое-то время, значит устройство не нашло нужную монету из общего количества монет - проставить галочку в служебном меню "Выдавать любыми". Либо перемешать монеты в хоппере, бывает засыпают номиналы слоями
  6. Если устройство не подключается из меню, при помощи програмы ITL включить/выключить/перезагрузить. При необходимости выкл/вкл порт хоппера из Диспетчера устройств
  7. При необходимости по инструкции выполнить чистку устройства
  8. При повторении случаев невыдачи, отправить устройство на сервисную чистку (платная)
  9. В случае застревания монеты (тестовая выдача не проходит), по инструкции достать монету
