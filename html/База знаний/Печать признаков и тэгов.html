Печать признаков и тэгов

Необходимо включить печать на чеке всех признаков и налогов:

  1. Закрыть ПО
  2. Запустить "ТестДрайвера"
  3. Открыть таблицы
  4. Таблица 1:
Поле 25 = 0 - Не использовать форматирования в чеке (если на чеке не печатается цена каждого товара)
  1. Таблица 17:
Поле 12 = 63 - печать всех реквизитов
Поле 16 = 2 - печатается только буква налога, ставка не печатается
Поле 17 = 2 - формат ФД 1.05
Поле 25 = 1 - печать автотэгов
Поле 39 = 1 - печать "Безналичные" вместо "Электронные"
  1. Перезагрузить кассу
  2. Закрыть "ТестДрайвера"
  3. Запустить ПО
