Не отображаются банковские слипы в ЛК

Банковский модуль СБЕРБАНК

При оплате банковской карте, в малом личном кабинете не отображаются банковские слипы.

1. Необходимо проверить версию прошивки устройства.

Просмотр версии прошивки:

Открыть папку C:\PC_STUB
Открыть на правку файл "pinpad.ini", раскомментировать (убрать знак ; ) строчку ;ShowScreens=1
Добавить строчку (если нет такой) PrinterFile=p
Сохранить и закрыть файл
Запустить файл "Loadparm.exe"
В появившемся всплывающем окне "UPOS.Главное меню", выбрать "Операции" -"Помощь".
Программа закроется и создаст файл "p"
Открыть файл "р" через Notepad++. В файлике будет указана прошивка модуля, версия операционной системы
Обязательно заново отредактировать файл "pinpad.ini" закоментировать строку ;ShowScreens=1
Сохранить и закрыть файл 29.76.00 - версия без поддержки слипов
29.92.00 и более новая - версия с поддержкой слипов

2. При 76й прошивке устройства, необходимо вызвать сотрудника Сбербанка для прошивки модуля.

3. При наличии параметра PrinterEnd=01 обязательно закоментировать его (поставить ; в начале строки)


Банковский модуль ГАЗПРОМ

При оплате банковской карте, в малом личном кабинете не отображаются банковские слипы.

1. Необходимо зайти в программу PosSSL.

- Выбрать 51. Терм. ИНФО - Выполнить
- В левом диалоговом окне найти внизу параметр "Сервер слипов" - должно быть значение "Выкл".

2. Если значение "Вкл." звонить Сергею из Газпрома.
